# Vrinda Store Annual Sales Report | MS Excel

## Project Overview

Created an interactive sales dashboard in Excel to analyze customer purchasing behavior, sales trends, and channel performance.

## Tools Used

* MS Excel
* Pivot Tables
* Pivot Charts
* Slicers

## Analysis Performed

* Monthly sales trends
* Gender-wise purchasing behavior
* Age group analysis
* Order status analysis
* State-wise performance
* Sales channel analysis

## Business Insights

Identified adult women customers as the major contributors to sales and recommended targeted marketing through Amazon, Flipkart, and Myntra platforms.
